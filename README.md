# Data Analyst Portfolio Project

## 📌 Project Overview

This project showcases data analysis and visualization skills using SQL, Python, Power BI, and Excel concepts. It focuses on understanding customer behavior and generating business insights from raw data.

---

## 📂 Files Included

* **Business Insights.sql**
  SQL queries used to extract, clean, and analyze data.

* **customer_behaviour_analysis.pbix**
  Interactive Power BI dashboard for visualizing key metrics and trends.

* **customer_shopping_behavior.csv**
  Dataset containing customer purchase and behavior details.

* **data_analyst_portfolio_project.ipynb**
  Jupyter Notebook with data cleaning, analysis, and visualization using Python.

* **README.md**
  Project documentation and overview.

---

## 🛠 Tools & Technologies

* SQL (Data Extraction & Querying)
* Python (Pandas, Matplotlib, Data Analysis)
* Power BI (Dashboard & Visualization)
* Excel (Data Handling Basics)

---

## 📊 Key Insights

* Identified customer purchasing patterns and trends
* Analyzed product performance and sales distribution
* Discovered customer segments based on behavior
* Generated actionable insights for business decision-making

---

## 🚀 How to Use

1. Open the `.sql` file in SQL Server to run queries
2. Load the `.csv` dataset into your preferred tool
3. Run the Jupyter Notebook for analysis steps
4. Open the `.pbix` file in Power BI to explore the dashboard

---

## 📈 Outcome

This project demonstrates the ability to transform raw data into meaningful insights using multiple data analysis tools and techniques.
